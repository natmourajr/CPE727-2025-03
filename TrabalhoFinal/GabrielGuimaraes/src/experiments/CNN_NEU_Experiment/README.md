# CNN_NEU_Experiment — CNN Baseline

## Descrição
Rede Neural Convolucional (CNN) baseline para classificação de defetos no dataset NEU, sem pré-treinamento.

## Pré-requisitos
- Python 3.10+
- PyTorch
- NumPy, Pandas, Matplotlib, Scikit-learn

## Instalação
```bash
cd src/experiments/CNN_NEU_Experiment
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Execução
```bash
python CNN.py
```

## Resultados
Os resultados são salvos em: **`src/Results/CNN_NEU/`**
