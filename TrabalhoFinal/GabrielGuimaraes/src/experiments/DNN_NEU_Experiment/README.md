# DNN_NEU_Experiment — DNN Baseline

## Descrição
Rede Neural Profunda (DNN) baseline para classificação de defetos no dataset NEU, sem pré-treinamento.

## Pré-requisitos
- Python 3.10+
- PyTorch
- NumPy, Pandas, Matplotlib, Scikit-learn

## Instalação
```bash
cd src/experiments/DNN_NEU_Experiment
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Execução
```bash
python DNN.py
```

## Resultados
Os resultados são salvos em: **`src/Results/DNN_NEU/`**
