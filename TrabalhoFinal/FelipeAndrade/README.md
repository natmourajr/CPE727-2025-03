# Classificação de Criadouros de Aedes aegypti no MBGv2 com CNNs e Autoencoders

Este repositório contém o código e os experimentos para comparar diferentes abordagens de aprendizado profundo na **classificação de potenciais criadouros de Aedes aegypti** a partir do dataset **MBGv2 (Mosquito Breeding Grounds Dataset v2)**, originalmente anotado para detecção de objetos em vídeos de drone.


## Requisitos

- Python 3.10+  
- Bibliotecas principais:
  - `torch`, `torchvision`
  - `numpy`, `pandas`
  - `scikit-learn`
  - `matplotlib`, `seaborn` (opcional para visualização)
- Demais dependências listadas em `requirements.txt`.
